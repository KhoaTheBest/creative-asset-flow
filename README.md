# POC React Flow

This project bootstraps a React Flow playground with Vite, React 18, TypeScript, Tailwind CSS, and ESLint.

## Requirements

- Node.js 20.10.0 (`nvm use` will pick it up from `.nvmrc`)
- Yarn 1.22.22

## Getting Started

```bash
yarn install
yarn dev
```

Navigate to http://localhost:5173 to explore the interactive flow chart.

## Available Scripts

- `yarn dev` – start the Vite development server
- `yarn build` – type-check and build the application
- `yarn lint` – run ESLint across the project
- `yarn preview` – preview the production build locally

## Tooling

- [React Flow](https://reactflow.dev/) for graph visualisation and editing
- [Tailwind CSS](https://tailwindcss.com/) for utility-first styling
- [ESLint](https://eslint.org/) ^8 with TypeScript support for static analysis
